Welcome to the **playwright-recast** documentation.

## What is playwright-recast?

playwright-recast is a fluent pipeline library that transforms Playwright trace files into polished demo videos — complete with TTS voiceover, styled subtitles, speed control, click effects, animated zoom, cursor overlay, and more.
